# Piper 1 (GPL)

A self-contained version of [Piper][piper], the fast and local neural text-to-speech engine.

This version directly embeds [espeak-ng][] and therefore has a [GPL license](COPYING).

Install with:

``` sh
pip install piper
```

[Listen to voice samples](https://rhasspy.github.io/piper-samples)

[Run directly in your browser](https://rhasspy.github.io/piper-samples/demo.html)

Usage:

* [Voices][voices]
* [Command-line interface][cli]
* [Web server][api-http]
* [Python API][api-python]
* [Training new voices][training]
